Podemos generar una funcin simple (en tiempo discreto) para proyectar el tamao poblacional futuro incorporando los parmetros del modelo logstico.

dlogistic <- function(K = 100, r = 1, N0 = 2, t = 15) {
    N <- c(N0, numeric(t))
      for (i in 1:t) N[i + 1] <- {
          N[i] + r * N[i] * (1 -  N[i]/K)
      }
          return(N)
}

Podemos reemplazar los parmetros con valores predeterminados para generar una proyeccin en el tiempo del tamao poblacional y graficarlos:

Diagramas de tela de araa

Una forma de visualizar la estabilidad de un modelo poblacional es a travs de un diagrama de tela de araa, el cual considera el tamao actual de la poblacin en el eje X y el tamao en el tiempo siguiente en el eje Y.


out <- dlogistic(N0=2, r=2.8, K=100, t=1000)
n <- out[901:1000]
ntp1 <- n[-1]
nt <- n[-length(n)]
qplot(nt, ntp1, geom=c("point", "path") ) + labs(y="N[t+1]", x="N[t]")+geom_abline(intercept = 0, slope = 1, color = "red")
